Basic Code Block

Add this structure to display code:

                
<pre class="language-javascript">
    <code>
        const example = "Hello World";
        console.log(example);
    </code>
</pre>

Live Example:

                
const example = "Hello World";
console.log(example);

Copy Button Feature

The copy button is automatically added to all code blocks.

                
// Copy button appears on hover
const data = {
    title: "Example",
    description: "Click the copy button!"
};

JavaScript Implementation:

                
import { CodeBlock } from './codeBlock.js';

// Initialize copy buttons
document.addEventListener('DOMContentLoaded', () => {
    new CodeBlock();
});

Line Numbers

Add line numbers to code blocks:

                
// Line numbered example
function example() {
    const items = [];
    for (let i = 0; i < 3; i++) {
        items.push(i);
    }
    return items;
}
