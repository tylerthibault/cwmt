Basic Modal Usage

Add this HTML structure:


<button class="openModalBtn" data-modal-target="myModal">Open Modal</button>

<div id="myModal" class="modal-wrapper">
    <div class="modal-content">
        <span class="close-modal">×</span>
        <div class="modal-header">
            <h3 class="modal-title">Modal Title</h3>
        </div>
        <div class="modal-body">
            Modal content here
        </div>
        <div class="modal-footer">
            <button class="close-modal">Close</button>
        </div>
    </div>
</div>

Advanced Features

Initialize with JavaScript:


import { Modal } from './modal.js';

const modal = new Modal('modalId');
modal.setContent('Dynamic content');
modal.open();

Methods Available:

  • open(): Open modal
  • close(): Close modal
  • toggle(): Toggle modal state
  • setContent(content): Update modal content

Event Handling


const modal = document.getElementById('myModal');

modal.addEventListener('modal:open', () => {
    console.log('Modal opened');
});

modal.addEventListener('modal:close', () => {
    console.log('Modal closed');
});

Customizing Modal Styles


.modal-wrapper {
    /* Overlay styles */
    background-color: rgba(0, 0, 0, 0.5);
}

.modal-content {
    /* Content box styles */
    max-width: 500px;
    border-radius: 8px;
}
