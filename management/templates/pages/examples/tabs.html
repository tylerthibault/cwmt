Basic Tab Implementation

Add these files to your project:


├── static/
│   ├── css/framework/tabs.css
│   └── js/framework/tabs.js

Basic HTML Structure:


<div class="tab-container">
    <div class="tab-nav">
        <button class="tab-btn active" data-tab="tab1">Tab 1</button>
        <button class="tab-btn" data-tab="tab2">Tab 2</button>
    </div>
    <div class="tab-content">
        <div class="tab-pane active" id="tab1">Content 1</div>
        <div class="tab-pane" id="tab2">Content 2</div>
    </div>
</div>

Form Navigation

Create multi-step forms with tab navigation:


<div class="tab-container">
    <div class="tab-nav">
        <button class="tab-btn active" data-tab="step1">Step 1</button>
        <button class="tab-btn" data-tab="step2">Step 2</button>
    </div>
    <div class="tab-content">
        <div class="tab-pane active" id="step1">
            <input type="text" name="username">
            <button class="next-tab" data-next="step2">Next</button>
        </div>
        <div class="tab-pane" id="step2">
            <input type="email" name="email">
            <button class="prev-tab" data-prev="step1">Back</button>
        </div>
    </div>
</div>

Advanced Features

JavaScript Initialization:


import { TabManager } from './framework/tabs.js';

document.addEventListener('DOMContentLoaded', () => {
    const tabs = new TabManager(document.querySelector('.tab-container'));
});

Available Methods:

  • switchTab(tabId): Switch to specific tab
  • getCurrentTab(): Get active tab ID
  • saveState(tabId): Save form state
  • loadState(tabId): Load saved state
